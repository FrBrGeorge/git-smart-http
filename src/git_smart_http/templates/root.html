How to Use

Create a New Repository

To create a new repository, simply clone a non-existent path from a trusted address:

git clone http://127.0.0.1:{port}/my-new-repo
cd my-new-repo
# Add files, commit, and push back
git push -u origin main

Clone an Existing Repository

From any address:

git clone http://{local_ip}:{port}/repository-name

From a trusted address (allows pushing back):

git clone http://127.0.0.1:{port}/repository-name

Server IP: {local_ip} | Port: {port}
